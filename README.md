Добро пожаловать в официальный репозиторий релизов **PokerBot**.

> **Важно:** Этот репозиторий используется **исключительно** для автоматической доставки обновлений (Auto-Updater) и хранения скомпилированных релизов. Исходный код бота является коммерческой тайной и закрыт.

## О проекте
**PokerBot** — это профессиональный инструмент для игры в покер, использующий передовые алгоритмы компьютерного зрения (OpenCV + SVM) и математического анализа (Monte Carlo) для расчета эквити и подсказок оптимальных ходов в реальном времени.

### Основные возможности:
- 🚀 **Автоматическое распознавание карт:** Мгновенный захват экрана и распознавание стола с высокой точностью.
- 🧮 **Быстрый расчет Эквити:** Вычисление шансов на победу за миллисекунды.
- 🛡️ **Система безопасности:** Полная привязка к железу (HWID) и защита от читерства со временем.
- 🔄 **Автообновления:** Клиент всегда поддерживает актуальную версию "по воздуху".

## Как установить
1. Перейдите на вкладку [Releases](../../releases).
2. Скачайте последнюю версию (файл `PokerBot_Setup.exe` или `update.zip`).
3. Запустите установщик.
4. Введите ваш лицензионный ключ в приложении.
