## О проекте
**PokerBot** — это профессиональный инструмент для игры в покер, использующий передовые алгоритмы компьютерного зрения и математического анализа для расчета эквити и подсказок оптимальных ходов в реальном времени.

### Основные возможности:
- 🚀 **Автоматическое распознавание карт:** Мгновенный захват экрана и распознавание стола с высокой точностью.
- 🧮 **Быстрый расчет Эквити:** Вычисление шансов на победу за миллисекунды.
- 🔄 **Автообновления:** Клиент всегда поддерживает актуальную версию.

## Как установить
1. Перейдите на вкладку [Releases](../../releases).
2. Скачайте последнюю версию (файл `PokerBot_Setup.exe` или `update.zip`).
3. Запустите установщик.
4. Введите ваш лицензионный ключ в приложении.

## 🔒 Безопасность и Антивирусы
Из-за специфики работы программы (захват экрана, чтение нажатий клавиш, использование оверлея) некоторые антивирусы (включая Windows Defender) могут ошибочно реагировать на бота (False Positive).

Мы гарантируем полную безопасность нашего софта. Программа не содержит вредоносного кода и собирает только необходимые для работы данные (HWID).
Вы можете ознакомиться с результатами проверки исполняемого файла независимым сканером:
 **[Результаты проверки на VirusTotal (2/69)](https://www.virustotal.com/gui/file/ac5eb5db62ba772a6f5cce98765f034d28ea348384f12f9f7d9687294de24099/behavior)**

> **Примечание:** Если Windows SmartScreen выдает синее окно при запуске ("Система Windows защитила ваш компьютер"), нажмите **Подробнее -> Выполнить в любом случае**. Если антивирус блокирует программу, добавьте папку с ботом в исключения.
